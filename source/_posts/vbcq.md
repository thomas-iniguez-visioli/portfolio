**vbcq** est l'écosystème web que j'ai développé et maintenu pour le **Volley Ball Corpo Quimper** — une association sportive locale. Ce projet regroupe plusieurs réalisations allant du site vitrine à des plugins et outils internes.

## Périmètre du projet

Le projet VBCQ couvre plusieurs aspects :

- **Site vitrine** de l'association, généré avec Hexo et hébergé via GitHub Pages
- **Plugins Hexo** sur-mesure pour les besoins spécifiques de l'association
- **Outils d'administration** pour faciliter la gestion du contenu par les membres non-techniques
- **Infrastructure** : automatisation des déploiements, synchronisation des dépôts

## Ce que j'ai livré

- Mise en place de l'organisation GitHub `vbcq-volley` avec plusieurs dépôts
- Site principal déployé sur [vbcq.vercel.app](https://vbcq.vercel.app/) (Svelte)
- Site vitrine statique déployé sur GitHub Pages
- Plugins JavaScript pour étendre les fonctionnalités Hexo
- Système de mise à jour automatique (`app-updater`)

## Technologies

`Hexo` · `Svelte` · `JavaScript` · `CSS` · `GitHub Actions` · `Vercel` · `GitHub Pages`
